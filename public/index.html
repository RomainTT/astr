Welcome on A.S.T.R.

A.S.T.R. is a powerful archiving system, designed to help you find your data quickly and easily.

You can interact with the system thanks to the website, but also directly with the RESTful API.
A Python library has been created to help you interact with it. Search for repository lib-python-astr.
